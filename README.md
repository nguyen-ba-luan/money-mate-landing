# Money Mate Landing Page

Landing page and privacy policy for the Money Mate personal finance app.

## Tech Stack

- Pure HTML/CSS/JS (no build step)
- Tailwind CSS v3 (CDN)
- Google Fonts (Inter, Playfair Display, JetBrains Mono)
- GitHub Pages hosting

## Structure

```
money-mate-landing/
├── index.html              # Landing page
├── privacy-policy.html     # Privacy policy
├── css/styles.css          # Custom styles + animations
├── js/main.js              # Interactions (scroll, nav, counters)
└── images/                 # Logo, icon, assets
```

## Development

Open `index.html` in a browser. No build step required.

## Deployment

Hosted on GitHub Pages at: https://code4change.github.io/money-mate-landing/
